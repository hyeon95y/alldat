# rocket-raccoon


## Setup repository

```
> hooks/install.sh
> apt-get update
> apt-get install direnv
> echo 'eval "$(direnv hook bash)"' >> ~/.bashrc 
> source ~/.bashrc
> direnv allow
> curl -s https://packagecloud.io/install/repositories/github/git-lfs/script.deb.sh | bash
> apt-get install git-lfs
> git lfs install
> git-lfs pull
```
